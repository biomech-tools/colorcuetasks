Firestoreのセキュリティルールが原因でアクセスできません。
Firebase Console → Firestore → ルール → 以下に変更して「公開」してください：

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
